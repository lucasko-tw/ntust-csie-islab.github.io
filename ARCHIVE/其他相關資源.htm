OpenNet
openNet is an initiative to create a Europe and worldwide focal point for excellence in research and development for next generation software systems.
Specifically the project will support the integration of research work in the area of adaptive, dynamically composable software systems by focusing activities on

  • open testbed network combining web
  • GRID services
  • agent technology
  • workflow and orchestration techniques
