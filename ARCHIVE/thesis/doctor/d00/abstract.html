一個植基於灰關聯結構的整合式樣本學習方法

在機器學習中，樣本學習方法扮演著相當重要的角色。在此類的學習方法中，有幾個關鍵問題極為重要，包含空屬性值的處理、相似函數的選擇以及訓練樣本的選取及縮減等。在本論文中，我們首先提出一個植基於灰關聯分析的方法以預測分類問題中各樣本的空屬性值，在填補這些空屬性值後，進而處理分類問題。在本論文中，我們也提出一個以灰關聯結構為基礎的樣本學習方法，樣本間的相似性透過灰關聯分析來測定。另外，我們亦提出基於上述灰關聯結構所衍生的樣本選取方法，藉由分析樣本間的灰關聯結構以達縮減訓練樣本數的目的。依序整合以上所提方法後可成功地解決上述一般樣本學習方法中的問題。

An Integrated Instance-based Learning Approach based on Grey Relational Structure

Abstract

In recent years, instance-based learning (IBL) has gained great popularity in machine

learning. In IBL, a training set of labeled instances is first collected in the learning system. A

new, unseen instance is then classified according to its ‘nearest’ training instance or instances.

This dissertation focuses mainly on these three key issues in IBL, including missing attribute

values, similarity functions and partial memory learning. First, this study proposes a

grey-based nearest neighbor approach to handle incomplete data problems. Here, grey

relational analysis is employed to determine the nearest neighbors of an instance with missing

attribute values. Thus, the valid attribute values derived from these nearest neighbors are used

to predict those missing values. Experimental results indicated that the proposed approach

outperforms other alternative methods. Furthermore, the proposed method was evaluated using

five classification problems with incomplete data. Experimental results revealed that the

accuracy of classification can be maintained or even increased by applying the proposed

method beforehand.

In this dissertation, an instance-based learning approach based on grey relational structure

is also proposed. Grey relational analysis is used to precisely describe the entire relational

structure of all instances in a specific application domain. By using the above-mentioned grey

relational structure, new instances can be identified with high accuracy. Experiments

performed on forty application domains are reported to demonstrate the performance of the

proposed approach. It can be easily seen that the proposed approach yields high performance

over other methods that adopt one of Euclidean metric and the Value Difference Metric (VDM)

or both. Moreover, it can be easily seen that the proposal presented here can yield satisfactory

performance, compared to other classification algorithms.

Furthermore, this study also proposes a novel partial memory-learning approach based on

grey relational structure. Grey relational analysis is used to precisely describe the entire

relational structure of all instances in a specific application domain. By using the

above-mentioned grey relational structure, instances having no or few inward edges can be

eliminated prior to learning (i.e., these instances are rarely used in determining the class labels

of other instances and thus can be pruned in a learning system). Accordingly, a suitable training

subset with small size can be obtained for classifying new instances. Experiments performed

on five application domains are reported to demonstrate the performance of the proposed

approach. It can be easily seen that the proposed approach yields high performance with small

size of training set, compared with other existing pruning techniques.

Notably, the proposed partial-memory learning method is an extended version of the

above instance-based learning approach. In these two learning approaches, the relationships

among instances are determined according to the grey relational structure (That is, grey

relational analysis is used to determine the ‘nearness’ (or relationship) between two instances).

Meanwhile, when classification tasks involve incomplete data, the above grey-based nearest

neighbor approach for missing attribute value prediction will be applied in advance in both of

the above two learning approaches. As a result, an integrated instance-based learning approach

is proposed with capability of dealing with the above-mentioned problems in IBL, including

missing attribute values, similarity functions and partial memory learning.
