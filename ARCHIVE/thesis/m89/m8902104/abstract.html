HitRank: 以使用者為導向的探勘方法為基礎 之網頁排序方式

摘要

基於網際網路上所提供龐大的異質性資料而言，使用者需要搜尋引擎來幫助他們找尋所需要的資訊。一般的搜尋引擎對於使用者的查詢所傳回的搜尋結果，以需求角度來說，並沒有很明顯或是突出的效果呈現，因此，對於搜尋引擎而言，如何將網頁做排序是一個重要的議題。雖然目前一些搜尋引擎都有它們對於網頁排序的演算法，並且試圖增加搜尋結果的準確度，但是這些排序的演算法仍然與使用者真正的需求有一段的差距。基於這些原因，在本篇論文裡，我們提出一種新的網頁排序探勘方法，叫做HitRank，它是以使用者為導向的回饋方式為依據來做為搜尋結果的排序，基於此種的探勘排序方法，我們可以有很大的機會去貼近使用者對於尋找資訊的需求。 這種排序方法首先針對使用者在使用搜尋引擎時，將瀏覽的點選序列用一個有向圖來表示，然後再以一個評估方程式來計算這個有向圖裡每一個網頁的分數，以此做為網頁排序的依據。此種排序方式不僅僅是考慮到該網頁被使用者點選的次數，同時也考量使用者真正的搜尋行為。此種排序方法所產生網頁排序的效果可以分為兩種：分別是具時間性的搜尋結果與關聯式的搜尋結果。具時間性的搜尋結果指的是搜尋結果的排序會隨著時間區段的不同而不同；而關聯式的搜尋結果則是可以推薦給其他具語意相關的查詢字來做為其搜尋結果。除此之外，我們將這種網頁的排序方法運用到一個個人化搜尋入口網站(Personal Search Portal)的系統上面，這個系統也確能幫助使用者更快的找尋他們所欲尋找的資料，同時能夠增加搜尋結果的準確度

HitRank : Search Ranking based on Mining User-Oriented Feedback

Abstract

Since vast heterogeneous data provided on the Internet, web-visitors need search engines to find their required information. General-purpose search engines are able to return useful search results with respect to web-visitors' queries. Therefore, how to rank web pages is critical to search engines. Although some search engines use their proprietary algorithms to rank web pages and try to increase the precision of search results, they still have the difficulty to meet web-visitors' requirements. These difficulties include the lack of collaborative feedback and weak relationship between actual query and search results. Based on these reasons, in this thesis, we provide a new ranking method, named as HitRank, to include the user-oriented feedback in the ranking of search results. In this way, we can have higher opportunity to meet the web-visitors' requirements. This ranking method is based on constructing a directed graph referring to the navigation of click streams of web-visitors. An evaluative function is then applied to give a score to each search result clicked by web-visitors. It not only concerns with the times of a web page clicked by web-visitors, but also interests in the actual search behaviors of web-visitors. Based on HitRank approach, two types of search results, i.e. temporal and associated search results, will be given to web-visitors. The temporal search results present that different timeframes provide different search result orders. On the other hand, a web page with high HitRank score can provide an associated search results for other semantic-dependent search queries. Moreover, we develop a personal search portal based on proposed HitRank method. By this portal, web-visitors can find their required information quickly and improve the precision of search results.
